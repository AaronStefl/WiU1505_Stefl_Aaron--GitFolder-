Aaron Stefl: Usability Landing Page

Project Descriptions

For the B2 assignment I created a simple 4 page wire frame mostly consisting of input fields and submit buttons. It contains name, email, country, address, phone number, age, gender, username, and password fields. It also contains specific Design patterns which was separately annotated. I also created an icon for World Surf. Page 2 is the error page.


B3 consists of a details page that contains Destination Information, Destination Cam and Tips & Tricks expanded pages. Each page expands to show information within each page. Page 4 is the bookmark page to bookmark-preferred locations.

B4 consists of the home page with the breakdown of World Surf and a Home page for users to login or sign up.
